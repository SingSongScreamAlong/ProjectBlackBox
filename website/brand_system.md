# okboxbox Brand System

## Brand Overview

**Name:** okboxbox  
**Tagline:** "AI Systems for the Modern Racer."  
**Domain:** okboxbox.com

---

## Brand Voice

### Personality
- **Confident** — We know our technology works
- **Calm** — Precision under pressure
- **Analytical** — Data-driven, never guessing
- **Human-centered** — Technology that serves people

### Tone Guidelines
| Context | Tone |
|---------|------|
| Headlines | Bold, declarative, no fluff |
| Body copy | Clear, conversational, technical when needed |
| CTAs | Action-oriented, confident |
| Error states | Helpful, never blaming |

### Writing Principles
1. Lead with value, not features
2. Use active voice
3. Be specific over generic
4. Respect the reader's intelligence
5. No buzzwords — say what you mean

## Color Palette

### Primary Colors
| Name | Hex | Usage |
|------|-----|-------|
| Primary Black | `#000000` | Backgrounds, text |
| Primary White | `#FFFFFF` | Text on dark, backgrounds |

### Accent Colors
| Name | Hex | Usage |
|------|-----|-------|
| F1 Red | `#E10600` | PitBox accent, CTAs, highlights |
| Electric Blue | `#00A2FF` | ControlBox accent |

### Neutral Grays
| Name | Hex | Usage |
|------|-----|-------|
| Dark Gray | `#1A1A1A` | Secondary backgrounds |
| Medium Gray | `#2B2B2B` | Cards, borders |
| Light Gray | `#CCCCCC` | Subtle text, dividers |

### Color Usage Rules
- **PitBox pages**: Use F1 Red (`#E10600`) as accent
- **ControlBox pages**: Use Electric Blue (`#00A2FF`) as accent
- **Global pages**: Use F1 Red as default accent
- **Never** use both accents together on the same element
- **Maintain** high contrast ratios (4.5:1 minimum for text)

## Typography

### Font Stack

#### Headings
```
font-family: 'Eurostile', 'Helvetica Neue', sans-serif;
```
**Fallback:** If Eurostile unavailable, use Orbitron or Industry

#### Body Text
```
font-family: 'Inter', 'Helvetica Neue', -apple-system, sans-serif;
```

### Type Scale
| Element | Size | Weight | Line Height | Letter Spacing |
|---------|------|--------|-------------|----------------|
| H1 (Hero) | 64px / 4rem | 700 | 1.1 | -0.02em |
| H2 (Section) | 48px / 3rem | 700 | 1.2 | -0.01em |
| H3 (Card) | 32px / 2rem | 600 | 1.3 | 0 |
| H4 (Subhead) | 24px / 1.5rem | 600 | 1.4 | 0 |
| Body | 18px / 1.125rem | 400 | 1.6 | 0 |
| Small | 14px / 0.875rem | 400 | 1.5 | 0.01em |
| Button | 14px / 0.875rem | 700 | 1 | 0.04em |

### Button Typography
- All uppercase
- Letter-spacing: +4% (0.04em)
- Font-weight: 700

## Visual Style

### Theme
- **Clean F1 aesthetics** — precision engineering feel
- **Understated military efficiency** — functional, no decoration for decoration's sake
- **Dark mode first** — black backgrounds, high contrast

### Design Principles
1. **Precision** — Every element has purpose
2. **Hierarchy** — Clear visual order
3. **Space** — Generous whitespace, let content breathe
4. **Consistency** — Unified patterns across all pages

### Imagery Guidelines
- High-quality sim racing screenshots
- Technical diagrams when explaining features
- Avoid stock photos of people
- Dark, moody lighting preferred
- No over-saturated colors

### Iconography
- Line-style icons preferred
- 2px stroke weight
- Consistent sizing (24px base)
- Match accent color of current product context

## Logo Usage

### Primary Logo
- Text-based: "okboxbox" in Eurostile Bold
- All lowercase
- May include tagline on larger formats

### Minimum Clear Space
- Clear space = height of "o" on all sides

### Logo Don'ts
- Don't alter letterforms
- Don't add effects (shadows, gradients)
- Don't use unapproved colors
- Don't rotate or distort

---

## Motion & Animation

### Principles
- **Subtle** — Enhance, don't distract
- **Fast** — 200-300ms for transitions
- **Purposeful** — Animation should communicate state

### Standard Transitions
| Element | Duration | Easing |
|---------|----------|--------|
| Hover states | 200ms | ease-out |
| Page transitions | 300ms | ease-in-out |
| Modal entry | 250ms | ease-out |

---

## Accessibility

### Requirements
- WCAG 2.1 AA compliance minimum
- Keyboard navigation support
- Screen reader compatibility
- Reduced motion support

### Color Contrast
- Text on dark: minimum 4.5:1
- Large text: minimum 3:1
- Interactive elements: clear focus states
