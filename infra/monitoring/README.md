# PitBox Monitoring System

Complete $0-$20/month monitoring stack.

## Quick Start
```bash
cd infra/monitoring
docker-compose up -d
```

Access:
- Grafana: http://localhost:3000 (admin/admin)
- Prometheus: http://localhost:9090

## Cost: ~$3/month
- Prometheus/Loki/Grafana: $0 (self-hosted)
- Sentry: $0 (free tier)
- AI summaries: ~$3/month

See docs/monitoring/ for full documentation.
